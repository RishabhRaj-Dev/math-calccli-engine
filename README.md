# math-calccli-engine
# CLI INTERFACE CALCULATOR 
A repository for a math calculator program made in 13-14 hours. can handle trigonometric functions and log, log 10. also handles complex string parsing and conversion to float values(eg. 2pi/3 can be made as (2*3.14)/3. languages used are python ).
# Features
1.Can handle trash input from user without crashing
2.Has 22 calc functions.
3.User friendly trigonometric input like if entered in radians(choice to enter in radians and degrees too) user can enter 2pi/3 or variations of it and the system will answer still correctly.
4.You in some functions can during a single run add as many characters you want to add in the operations like in additions or multiplication or subtraction functions.
5.Menu based function input you can choose the menu number to select the function
# Installation and Usage
1. Clone the repository:

git clone https://github.com/RishabhRaj-Dev/math-calccli-engine.git

2. Navigate to the project folder:

cd "Project Python"
(Make sure to get the space between Python and Project and it is case sensitive!)

3. Make sure Python is installed on your system.
# Usage and Running
## Option 1- Terminal
1.In the project folder run:
python calccli.py
## Option 2-IDE
You can also run it on ide perform following after installation
1.Open IDE
2.Open file---> Folder
3.Select the folder Project Python
4.Select and run the calccli.py file
5. Click Run
